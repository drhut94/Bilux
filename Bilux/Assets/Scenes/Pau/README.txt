MECÁNICAS
Movimiento -> A,D.
Saltar -> Space
Turbo  -> Shift + A/D. El rastro azul que deja la bola es la cantidad de turbo que queda. El tubo se recarga con el tiempo.
Gancho -> Estar en la area del gancho (tienen forma de X). Cuando se esta en la area del gancho la X se pondra en verde.
       -> space para sacar el gancho, turbo + A/D para moverse, space otra vez para soltarse.

BLOQUES
Blancos -> Normales. Ninguna función en especial.
Verdes  -> Destructibles. Se destruyen al alcanzar una cierta velocidad.
Amarillos -> Dan un impulso de velocidad hacia donde indican las flechas.
Pinchos -> Te matan.
Agua -> Te quita vida periódicamente. Te mueves mas lento. Puedes saltar.

OTROS
Vida -> A medida que vas perdiendo vida, el mapa se irá volviendo en blanco y negro, así dificultando identificar de que color es cada bloque.
R -> Restart.
